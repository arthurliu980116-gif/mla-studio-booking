GitHub Pages

公开网页发布说明

这个站点是纯静态网页，适合发布到 GitHub Pages。Pages 版本中，每个访问者的数据保存在自己的浏览器，不会共享真实库存。

  1. 创建公开 GitHub 仓库，建议命名为 mla-studio-booking。
  2. 上传 github-pages-site/ 目录中的全部文件到仓库根目录。
  3. 进入 Settings -> Pages，Source 选择 Deploy from a branch。
  4. Branch 选择 main，Folder 选择 /(root)，保存。
  5. 等待 GitHub Pages 发布完成后，点击 Visit site。
